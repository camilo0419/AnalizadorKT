==============================
 ANALIZADOR DE CLÁUSULAS v3.1
==============================

📘 Descripción
---------------
Herramienta de escritorio para comparar los títulos de cláusulas 
entre un archivo PDF de póliza y una base de referencia en Excel.

Permite seleccionar el ramo que se desea analizar:
 • MRC (Multiriesgo Corporativo)
 • Transporte

El resultado se guarda automáticamente en un nuevo archivo Excel,
con la lista de cláusulas encontradas, no encontradas y observaciones.

🧭 Instrucciones de uso
------------------------
1. Copie o tenga en esta misma carpeta los siguientes archivos:
   - AnalizadorClausulas.exe
   - base_clausulas_mrc.xlsx
   - base_clausulas_transporte.xlsx
   - icono.ico

2. Ejecute **AnalizadorClausulas.exe** (doble clic).

3. Ingrese la contraseña: `**********`

4. Presione el botón del ramo correspondiente:
   - 📄 **Analizar MRC**
   - 🚚 **Analizar Transporte**

5. Seleccione el archivo PDF de la póliza.

6. Espere mientras se ejecuta el análisis.
   Al finalizar, el programa pedirá dónde guardar el resultado (.xlsx).

7. Abra el archivo generado para revisar las coincidencias.

📈 Requisitos
--------------
- No requiere instalación de Python.
- Windows 10/11 (64 bits).
